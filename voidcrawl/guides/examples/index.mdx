All examples live in the [`examples/`](https://github.com/CascadingLabs/VoidCrawl/tree/main/examples) directory of the VoidCrawl repository and can be run directly after building.

```bash
uv run examples/basic_navigation.py
```

## Available Examples

| Example | What it demonstrates |
|---------|---------------------|
| [Basic Navigation](/voidcrawl/guides/examples/basic-navigation/) | Launch headless, navigate, read title and content |
| [DOM Interaction](/voidcrawl/guides/examples/dom-interaction/) | `query_selector`, `click_element`, `type_into` |
| [JavaScript Eval](/voidcrawl/guides/examples/javascript-eval/) | Evaluate JS expressions, modify DOM, return structured data |
| [Screenshots](/voidcrawl/guides/examples/screenshots/) | Capture PNG screenshots |
| [Stealth Mode](/voidcrawl/guides/examples/stealth-mode/) | Compare fingerprints: stealth on vs. off |
| [Actions Demo](/voidcrawl/guides/examples/actions-demo/) | Built-in actions, custom JS actions, Flows, CDP actions |
| [Docker Headful](/voidcrawl/guides/examples/docker-headful/) | Connect to Docker Chrome with GPU + VNC |
| [Multi-Page](/voidcrawl/guides/examples/multi-page/) | Multiple tabs in a single session |

## More examples (run from the repo)

These additional examples live in the `examples/` directory but don't have dedicated doc pages yet. Run them directly:

```bash
uv run examples/cookies.py                     # Set, read, and delete cookies via CDP
uv run examples/network_logging.py             # Capture and inspect all network requests
uv run examples/custom_headers_and_proxy.py    # Set custom HTTP headers and proxy config
uv run examples/connect_to_existing_chrome.py  # Attach to already-running Chrome via ws_url
uv run examples/concurrent_stress_test.py      # 30 tasks across 3 pools with interactive search
uv run examples/debug_session.py               # Interactive step debugger (debug=True)
uv run examples/httpx_vs_voidcrawl.py          # Compare plain HTTP vs JS-rendered scraping
uv run examples/performance_modes.py           # Scale profiles and hardware-aware pool sizing
uv run examples/docker_headless.py             # Docker headless with PoolConfig.from_docker()
```
